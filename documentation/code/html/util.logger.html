get_logger(output_file: str, name: str = None, overwrite: bool = True) → Logger[source]

Creates a log of application activity.

Parameters:

  • output_file – the name of the output file to generate.

  • name – the name to give logger object. Defaults to None.

  • overwrite – whether to overwrite file if it already exists. Defaults to True.

Returns:

A logger object that records information to both the specified file and standard output.

Note

Output file should be a filename and will be stored in the logs directory. If there is file with a matching name, the logger will create one. If a file with that name already exists, it will be overwritten unless otherwise specified.
