class Timer(limit: int, task: callable, start_time: int)[source]

Bases: Timer

A mechanism for executing a set of instructions after waiting a specified amount of time.

The timer class is a child class of the threading.Timer class with minor added functionality. With every new object, a new thread is created, ensuring the specified task will execute regarldless of the main system thread.
