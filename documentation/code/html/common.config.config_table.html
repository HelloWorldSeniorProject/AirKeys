Config Table module

class ConfigTable(*args, **kwargs)[source]

Bases: object

A shared configuration for use across the system.

Configuration Table is a singleton class that contains information about the system’s state and other important configuration information. Utilizes thread locks to prevent concurrent access while modifying/retrieving table information.
