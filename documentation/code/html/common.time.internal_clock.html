Internal Clock module

class InternalClock(*args, **kwargs)[source]

Bases: object

A monotonic clock for use across the system.

Internal Clock is a singleton class that has functions for determining relational times throughout the system. Utilizes thread locks to prevent concurrent access to CPU hardware clocks.
